Here is all news of different 7 catagories such as trade, sports, quadruple,
national, international, frontpage and editorial.
There are two python scripts taht are used to scrap all these news.
1.news_link_collector_script.py
-->This script first collect all news links and write them in a csv file
2.data_collector_script.py
-->This script read each link of previously generated csv file and scrap 
-->the whole news, news title, news date and news catagory of that individual news link. 
-->Finally it stores all the data in a csv file.
